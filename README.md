
How to build:

On Linux

1. install cmake:
```
sudo apt-get install cmake
```

2. Under the RL folder, run 

```
mkdir build
cd build
cmake ..
make
./RL_demo

```


On Mac

1. install cmake:
```
brew install cmake
```

2. Under the RL folder, run 

```
mkdir build
cd build
cmake ..
make
./RL_demo

```
